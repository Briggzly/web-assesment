How does the internet work?

The internet is a worldwide network of local networks. There is an application built on top of the Internet called the web. The web is pretty much just interconnected system of public web pages accessible through the Internet. Networks are when 2 computers need to communicate they have to do that either physically or wirelessly. The Internet is like the country and the web is like the highways and roads connecting the cities and states.

What happens when you click on a link?

When you click on a link it sends an initial request to the server. The request then reaches the app server which then the server sends HTML to the browser and the browser begins processing. The HTML processing finishes the app code finishes execution and the page is rendered in the browser.
